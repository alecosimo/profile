Odin MBS code

Odin is a research code for the simulation of nonsmooth flexible multibody systems which heavily uses Lie group methods not only for the description of the motion, but also for the time integration and for the spatial discretization of flexible components. Odin will be available soon as opensource code under the Apache2 license.

I started this code from scratch at the beginning of 2020. It is written in C++17, however, I recently started to use some features of C++20. Some of the features that this code has:

      Lie group time integrators for smooth and nonsmooth problems Kinematic joints: all lower pairs Flexible elements: a 2-noded SE3 beam element and its n-noded version which allows you to choose any polynomial order Nonsmooth monolithic formulations for the contact between rigid and elastic bodies Contact between flexible beams Iterative Gauss-Seidel like solvers for nonsmooth problems available through the So-Bogus library Contact detection is available by using the Bullet library Python bindings available by using the pybind11 library
