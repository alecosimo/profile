A Global-Local formulation for dealing with problems with concentrated moving sources is proposed. A global coarse mesh is used for the global domain, whilst a fine mesh is used for the local domain which moves following the concentrated heat source. This technique has the advantage of reducing not only the number of degrees of freedom, but also of enabling the application of reduced order models for the fine domain.
