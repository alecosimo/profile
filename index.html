Projects

In what follows, you will find a brief description of the main projects in which I have been involved. Much of the ideas that I present were coded almost entirely by me as extensions of a pre-exiting large Finite Element code written in C++. Unfortunately, this code is proprietary software and I can not share the code I wrote. Nonetheless, if you want to take an idea of my coding abilities, you can have a look to the code found in my github account that I wrote for experimenting with FETI solvers (a family of linear equations solvers targeting distributed systems).
